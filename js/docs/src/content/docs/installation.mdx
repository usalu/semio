### 🧩 User Interfaces
:::note
semio currently supports two interfaces:
:::


<div style="display: flex; flex-direction: column; gap: 0.5rem;">

  <div style="display: flex; gap: 1rem; align-items: baseline;">
    <strong style="min-width: 120px;">🦗 Grasshopper</strong>
    <span>Rhino 8 plugin for creating and managing semio kits in a parametric workflow.</span>
  </div>

  <div style="display: flex; gap: 1rem; align-items: baseline;">
    <strong style="min-width: 120px;">✏️ Sketchpad</strong>

  </div>

</div>


<Tabs>
  <TabItem label="🦗 Grasshopper (Rhino 8)">

  :::tip[Interested in the latest version?]
  Then hit `Include pre-releases` and download `5.3.0-beta` ⬇️
  :::

  The Grasshopper plugin can be installed through the **Rhino Package Manager** 🟢

  ![Rhino Package](../../../../../assets/images/rhinopackage.gif)

  <Steps>
  1. Open Rhino Package Manager ⌨️  
  2. Search for **semio** 🔎  
  3. Install the latest version ⬇️  
  4. Restart Rhino 🔄  
  </Steps>

  That’s it!

  </TabItem>

  <TabItem label="✏️ sketchpad">

  :::caution[Still a prototype 🧑‍💻]
  It doesn’t use the latest versions of the engine and Grasshopper ⚠️
  :::

 [**Sketchpad**](https://github.com/usalu/semio/releases/download/r24.07-1/sketchpad.exe) is a **portable program** — no installation needed.
 
   
  ![sketchpad demo](../../../../../assets/images/sketchpad-demo.png)

 

  - To use Sketchpad **with Grasshopper**, download **semio Grasshopper `2.1.2`**.  
  - To use Sketchpad **standalone**, download, extract, and run the [**engine**](https://github.com/usalu/semio/releases/download/r24.07-1/engine.zip).  
  - You can then explore the [**examples**](https://github.com/usalu/semio/releases/download/r24.07-1/examples.zip) 🏎️

  </TabItem>
</Tabs>
