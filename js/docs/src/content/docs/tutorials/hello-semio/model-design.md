### 🆔 Name Piece

### 🧬 Link Mold

Reference a mold and variant to create a piece.

## 🔗 Make Connections

### 🔀 Set Connection

Define how bricks snap together via Ports.

### 🎯 Adjust Snap

Move or rotate connected bricks for precise placement.
